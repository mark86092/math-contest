## 第一題

設 $n$ 與 $p$ 都是正整數。試證：若多項式 $f(x) = x^n + x^{n-1} + \cdots + x + p$ 可以分解為兩個次數不小於 $1$ 的整係數多項式之乘積，則 $p$ 不是質數。

## 第二題

已知凸四邊形 $ABCD$ 的周長為 $10$，對角線 $\overline{AC} = \overline{BD} = 4$。試問四邊形 $ABCD$ 最大可能的面積為多少？

## 第三題

設 $n$ 為正整數。今有 $2n$ 張紙牌，分別寫上號碼 $1, 2, 3, \ldots, 2n$。將這些紙牌排成一列：$a_1, a_2, \ldots, a_{2n}$，不妨將此種排列記為 $S$。對於此種牌型 $S$ 中的 $i$ 號牌和 $2n+1-i$ 號牌中間所夾的紙牌張數記為 $f_i$，並令

$$|S| = f_1 + f_2 + \cdots + f_n.$$

例如，當 $n = 2$ 且排列 $S: 1324$，則 $f_1 = 2, f_2 = 0$，因此 $|S| = f_1 + f_2 = 2$。

回答以下的問題：

1. 證明：對於任何排列 $S$，$|S|$ 必為偶數。
2. 給定 $n$，證明 $|S|$ 的最大值為 $n(n-1)$。並求出滿足 $|S| = n(n-1)$ 的所有可能的 $S$ 的數目。
