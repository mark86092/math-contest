## 填充題

每小題 5 分，共 60 分。

1. 設 $x, y$ 均為非負整數且滿足 $\sqrt{x} + \sqrt{y} = \sqrt{11300}$，求滿足上式的 $x + y$ 共有多少個不同的值？

2. 若數 $(3630)_p$ 能被 7 整除，求所有符合條件的 $p$ 值為何？（例如：$(1011)_2 = 1 \times 2^3 + 0 \times 2^2 + 1 \times 2^1 + 1 \times 2^0$；$(310)_5 = 3 \times 5^2 + 1 \times 5^1 + 0 \times 5^0$）

3. 假設 $x$ 的方程式 $x^2 + tx - 3 = 0$ 和 $x^2 - 4x - (t-1) = 0$ 有一個相同的根，則此根為何值？

4. 有一個 2026 個數的數列，首項為 1，除了首項和末項外，每項之值等於其前後項之和，試求此數列末項的值為何？

5. 高雄市某國中有 65 位學生參加教育局舉辦的 $A, B, C, D$ 四種項目競賽，其中有 45 人參加 $A$ 項目，48 人參加 $B$ 項目，52 人參加 $C$ 項目，55 人參加 $D$ 項目，問至少有幾個人四個項目均參加？

6. 設 $m$ 為整數，且 $12 < m < 40$，若方程式 $x^2 - 2(2m-3)x + 4m^2 - 14m + 8 = 0$ 有兩個整數根，求 $m$ 之值。

7. 已知 $2026 < n < 2030$，設 $M = 2021^n + 2022^n + 2023^n + 2024^n + 2025^n$，若 $n$ 滿足 $M$ 能被 5 整除，求 $n$ 之值？

8. 設 $x, y, z$ 為正數，且 $xyz(x + y + z) = 196$，求 $(x+y)(y+z)$ 的最小值為何？

9. 如圖所示，已知正方形 $ABCD$ 的邊長為 6，點 $O$ 在 $\overline{AB}$ 邊上，且 $\overline{OB} = 2$，若 $P^\prime$、$Q^\prime$ 分別為 $\overline{BC}$、$\overline{CD}$ 上的動點，但不在這兩邊的端點位置，求四邊形 $AOP^\prime Q^\prime$ 的最短周長為何？

10. 直角三角形 $ABC$ 的三個頂點 $A$、$B$ 與 $C$ 均在二次函數 $y = kx^2$ 的圖形上，已知 $k$ 為正數，且斜邊 $\overline{AB}$ 與 $x$ 軸平行。試求斜邊 $\overline{AB}$ 上的高 $h$ 之值（答案以 $k$ 的形式表之）。

11. 如果把數字 $1, 2, 3, 4, 5, 6$ 各用一次，組成兩個三位數 $m$ 和 $n$，求 $m \times (n + 70)$ 之最大值為何？

12. 如圖所示，直角三角形 $ABC$，已知 $\overline{BD} = 14$，$\overline{CD} = 16$，且 $2\angle ADB = 3\angle ACB$，試求 $\triangle ACD$ 之面積。

## 第一題

設 $a$、$b$ 與 $c$ 均為任意數（正數、負數或零），已知 $abc = 21$，$a(b+1)(c+1) = 45$ 及 $a(b+2)(c+2) = 77$。試求出 $a(b-1)(c-1)$ 的值。

## 第二題

設 $a$、$b$ 是方程式 $x^2 - 3x - 5 = 0$ 的兩根，$c$、$d$ 是方程式 $x^2 - 2x - 2 = 0$ 的兩根，則

$$\frac{25a - a^3}{b + c + d} + \frac{25b - b^3}{a + c + d} + \frac{25c - c^3}{a + b + d} + \frac{25d - d^3}{a + b + c}$$

之值為何？
